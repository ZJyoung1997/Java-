# Java图书管理系统

该Java 图书管理系统实现了借书、还书、查询图书、管理员后台管理等功能。

###### 将Java项目做成不需要JDK就能运行的安装包详细教程：https://blog.csdn.net/k_young1997/article/details/83349568
项目运行后会自动生成数据库。
